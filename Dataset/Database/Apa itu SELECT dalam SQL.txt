SELECT adalah perintah SQL yang digunakan untuk mengambil data dari sebuah database.
Pernyataan SELECT digunakan untuk melakukan kueri pada database dan mengambil data yang sesuai dengan kriteria yang ditentukan.
SELECT memungkinkan Anda untuk menentukan kolom-kolom yang ingin diambil dari sebuah tabel.
SELECT * mengambil semua kolom dari tabel yang ditentukan.
Pernyataan SELECT dapat menyertakan kondisi untuk menyaring data menggunakan klausa WHERE.
SELECT dapat mengambil data dari satu atau lebih tabel menggunakan JOIN.
Anda dapat menggunakan SELECT untuk melakukan perhitungan dan mengembalikan hasil yang dihitung.
SELECT DISTINCT digunakan untuk mengambil nilai-nilai unik, menghilangkan duplikasi dari hasil.
Klausa ORDER BY dalam SELECT digunakan untuk mengurutkan data yang diambil dalam urutan naik atau turun.
Klausa LIMIT dalam SELECT membatasi jumlah baris yang dikembalikan oleh kueri.
SELECT memungkinkan penggunaan fungsi agregat seperti COUNT, SUM, AVG, MAX, dan MIN untuk melakukan perhitungan pada data.
Klausa GROUP BY dalam SELECT digunakan untuk mengelompokkan baris yang memiliki nilai yang sama dalam kolom tertentu menjadi baris ringkasan.
Klausa HAVING dalam SELECT digunakan untuk menyaring kelompok berdasarkan kondisi, mirip dengan klausa WHERE tetapi untuk kelompok.
SELECT dapat digunakan dengan subkueri untuk mengambil data berdasarkan hasil dari kueri lain.
Anda dapat menggunakan kata kunci AS dalam SELECT untuk mengganti nama kolom atau membuat alias.
SELECT memungkinkan penggunaan logika kondisional dengan pernyataan CASE.
Anda dapat menggabungkan beberapa kueri SELECT menggunakan UNION atau UNION ALL untuk mengembalikan satu set hasil.
SELECT mendukung penggunaan pencocokan pola dengan operator LIKE.
SELECT dapat mengambil data dari views, yang merupakan tabel virtual berdasarkan hasil dari kueri SELECT.
Pernyataan SELECT INTO digunakan untuk membuat tabel baru dan memasukkan hasil ke dalamnya.
